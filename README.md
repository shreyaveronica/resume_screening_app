# resume_screening_app
# Intelligent Resume Screening and Ranking System

## Overview

The increasing competitiveness in the job market has made manual resume screening inefficient, time-consuming, and prone to inconsistency. This project presents an intelligent resume screening system that automates candidate evaluation using Natural Language Processing (NLP) and Machine Learning techniques.

The system processes resumes, matches them with job descriptions, ranks candidates based on relevance, and classifies them into different suitability levels.

---

## Key Features

- Multi-format resume support (PDF, DOCX, TXT)
- Text preprocessing and normalization
- TF-IDF based feature extraction
- Cosine similarity for resume ranking
- Logistic Regression for candidate classification
- Interactive web dashboard using Streamlit
- Filtering and search functionality

---

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas and NumPy
- PDFMiner (PDF parsing)
- python-docx (DOCX parsing)
- Basic CSS (UI styling)
